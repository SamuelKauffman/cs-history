Significant Contribution

In 1993 he developed Wine a software program that enables users to run many Microsoft Windows applications under Linux. Creating a file manager a software program that helps a user manage all the files on computers. In 1997 along with Federico Mana, Miguel began to contribute, to the development of free operating systems. In the year 2001, Miguel De Icaza led the mono project whose purpose was to implement Microsoft's new net development platform. Miguel de Icaza built a Net runtime for Android and iOS as well as a cloud testing platform (Xamarin Test Cloud). Icaza endorsed the Microsoft Office Open XML document, where he disagreed with a lot of criticism regarding free open sources in the software community.

Biography

Miguel de Icaza is a Mexican programmer best known for starting the Gnome, Mono, and Xamarin projects. He was born on November 23, 1972 in Mexico City. He is currently 50 years old. Miguel attended the National Autonomous University of Mexico where he studied Mathematics which he did not finish. He began writing software in the year 1992. At Novell, Miguel was VP for the Developer Platform. At Microsoft, He was a Distinguished Engineer working on Xamarin and . NET. He loved working with the .NET team and the C# language design team.
